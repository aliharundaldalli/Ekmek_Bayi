# Katkıda Bulunma Rehberi

Projeye katkıda bulunmak istediğiniz için teşekkürler! Aşağıdaki adımları takip ederek katkıda bulunabilirsiniz.

## Nasıl Katkıda Bulunabilirim?

1.  **Forklayın**: Bu repoyu kendi GitHub hesabınıza forklayın.
2.  **Klonlayın**: Forkladığınız repoyu bilgisayarınıza indirin.
3.  **Branch Oluşturun**: Yapacağınız değişiklikler için yeni bir branch oluşturun.
    ```bash
    git checkout -b yeni-ozellik
    ```
4.  **Değişiklik Yapın**: Kodunuzu yazın ve test edin.
5.  **Commitleyin**: Değişikliklerinizi açıklayıcı bir mesajla commitleyin.
    ```bash
    git commit -m "Özellik: Yeni giriş sayfası tasarımı eklendi"
    ```
6.  **Pushlayın**: Branchinizi GitHub'a gönderin.
    ```bash
    git push origin yeni-ozellik
    ```
7.  **Pull Request (PR) Açın**: GitHub üzerinden ana repoya Pull Request gönderin.

## Kodlama Standartları

*   PHP kodu için PSR-12 standartlarına uymaya çalışın.
*   Değişken ve fonksiyon isimlendirmelerinde İngilizce veya Türkçe tutarlı bir dil kullanın.
*   Yorum satırları eklemeyi unutmayın.

## Hata Bildirimi

Bir hata bulursanız, lütfen GitHub Issues bölümünden detaylı bir şekilde bildirin.
